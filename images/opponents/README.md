# 對手圖片放這裡

把藥師少女的獨語角色圖片放到這個資料夾，檔名要對：

| 角色 | 檔名 | 顯示位置 |
|---|---|---|
| 玉葉妃 | `gyokuyou.png` | 初級對手 |
| 高順 | `gaoshun.png` | 中級對手 |
| 壬氏 | `jinshi.png` | 高級對手 |
| 貓貓 | `maomao.png` | 頂級對手 |

格式建議：
- **正方形**（會被裁切成圓形）
- 200x200 ~ 400x400 像素都可
- PNG 或 JPG 皆可
- 檔案大小建議 < 200 KB

若沒有放圖片，App 會自動顯示對應 emoji（🌸/🧔/🎭/🦊）作為 fallback。
